# Hyper OS Brand Identity

## Name
- Product name: **Hyper OS**

## Tagline
- **"Fast. Focused. Game Ready."**

## Design philosophy
- Minimal visual noise
- Fast UI rendering paths
- Futuristic accent treatment
- Gaming-ready utility first

## Palette
- Background: `#0B0F14`
- Surface: `#121A24`
- Primary text: `#E6EDF7`
- Secondary text: `#9BB0C8`
- Accent: `#29D3FF`
- Accent alt: `#7C5CFF`

## Typography
- UI: Noto Sans / DejaVu Sans fallback
- Terminal: DejaVu Sans Mono

## Icon direction
- Flat, geometric, neon-accent monochrome
- Avoid skeuomorphic/high-detail icon sets
