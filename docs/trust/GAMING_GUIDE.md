# Hyper OS Gaming Guide

## Steam + Proton
1. Install/launch Steam.
2. Enable Steam Play + Proton in settings.
3. Use per-game profile wrapper in launch options.

## Windows game launching
- Steam: `hyperos-game-launch --profile <profile> -- %command%`
- Lutris prefix: `hyperos-game-launch --profile lutris-default --`

## Validate gaming stack
```bash
hyperos-game-validate
```
