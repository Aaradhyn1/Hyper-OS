# Hyper OS Getting Started

## Minimum requirements
- 64-bit CPU with virtualization support recommended
- 4 GB RAM minimum (8 GB recommended)
- 35 GB disk for installed system
- UEFI or Legacy BIOS supported

## Install steps
1. Download ISO and SHA256 file from official release page.
2. Verify checksum before writing USB.
3. Boot USB and choose Hyper OS live mode.
4. Launch installer from desktop and complete Calamares flow.
5. Reboot into installed system.
